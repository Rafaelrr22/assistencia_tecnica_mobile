# Assistência Técnica Mobile

Aplicação Android para gestão de assistências técnicas.

## Funcionalidades
- Registo de clientes
- Registo de equipamentos
- Registo de assistências
- Histórico de intervenções
- Fotografias
- Exportação PDF

## Tecnologias
- Kotlin
- Jetpack Compose
- SQLite / Room
- Android Studio
